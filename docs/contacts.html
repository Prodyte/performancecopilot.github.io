Contacts


Mailing List

Questions and discussion about the toolkit should be directed to the PCP mailing list, https://groups.io/g/pcp .

IRC Channel

Several PCP developers and fellow users can be contacted via the #pcp channel on the Freenode network.

Web pages

The PCP web pages at https://pcp.io/ cover PCP, PCP GUI and PCP Glider (the native Windows distribution of PCP).

Binaries

Pre-built packages for Redhat/SuSE, Debian/Ubuntu, Windows, MacOSX and Solaris can be downloaded from the PCP website at https://bintray.com/pcp/ .

Source Code

GIT: https://github.com/performancecopilot/pcp/ .

Compressed source tarballs can be found alongside the binary packages above.
